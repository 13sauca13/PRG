## Qué es el big data?
El Big Data son volúmenes de datos masivos que no pueden ser tratados mediante técnicas convencionales:
+ Captura
+ Transferencia
+ Almacenamiento
+ Gestión/mantenimiento/consulta
+ Análisis
+ Visualización
